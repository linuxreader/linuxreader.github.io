Create connection between client and server

Enable Podman service on the server:

systemctl --user enable podman.socket

Enable lingering for your user:

sudo loginctl enable-linger $USER

Make sure qemu-img is installed: sudo dnf install qemu-img sudo dnf -y install qemu-img.x86_64

Install gvproxy sudo dnf install -y podman-gvproxy

Initialize and start a Podman machine: (this may take some time)

podman machine init
podman machine start

Verify the installation:

podman info

Verify the socket is listening:

podman --remote info
